This work sets out to investigate detecting the driver from previously introduced objective criteria, which describe the driving style of race cars drivers. The original dataset is expanded to three full race seasons. A supervised learning approach using Random Forest is tested on different variants of the available data: Direct use of the corner specific observations, averaged values for all features summarising all corners within a single lap as well as a majorityvote of all predictions from the corners within a lap. Additionally a variation of hyperparameters for the used classifier is conducted. The best classification is seen for the majority vote method in combination with 500 trees in the forest, a minimal node size of two and ten possible splitting variables per node with an accuracy value of 95.9% on the test set.
